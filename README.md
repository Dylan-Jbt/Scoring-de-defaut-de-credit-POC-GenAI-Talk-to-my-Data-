# Scoring-de-d-faut-de-cr-dit-POC-GenAI-Talk-to-my-Data-
Conception d’un modèle de scoring du risque de crédit et d’un POC GenAI pour l’analyse de données en langage naturel, avec mise en place d’une pipeline ML end-to-end et d’une interface interactive avec Python (pandas, scikit-learn, matplotlib, plotly, langchain, langfuse, streamlit).
